\subsection{Detailed Description}
Class for rendering the border of a \doxyref{SlimWindow}{p.}{class_sonetto_1_1_slim_window}. \begin{Desc}
\item[Remarks:]We need this because we have to render twice, once with the inner panel's repeating material (handled by superclass) and once for the border's separate meterial. \end{Desc}
